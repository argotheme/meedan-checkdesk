translationUI (service in module cd )

Description

The cd.translationUI module houses the service and controller necessary to manage the Checkdesk real-time translation interface.

Methods

  • config()

    Get's a reference to $translate's master translationTable. This is sort of a hack but is a very handy way to directly access the current list of translated strings on the page.

    See: https://github.com/PascalPrecht/angular-translate/pull/61
